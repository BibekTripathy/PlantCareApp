# PlantCareApp
Work-in-progress Plant Care App

## Build
### Dependencies
- qt6
- sqlite3

### Steps
```sh
# Clone the repository
git clone https://github.com/BibekTripathy/PlantCareApp.git

# Create build directory
mkdir build
cd build

# Configure cmake
cmake ..

# Build the program
cmake --build .

# Run the program
./bin/PlantCareApp
```
